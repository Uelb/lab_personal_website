The superSQL project


SuperSQL is an extension of the SQL language which allows you to easily and quickly structure the output of a relational database. Moreover, this language provides a large variety of layouts.


The SELECT clause of the SQL language is here replaced by the GENERATE clause. The syntax is GENERATE where the media is the ouptut language of the query and TFE is the Target Form Expression. For instance, the media can be HTML, or PDF, SWF, or Excel format. TFE represents the way the considered list will be displayed (which operator to choose).
